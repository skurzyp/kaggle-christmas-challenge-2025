# Go Solution

This directory contains the Go implementation for the Tree Packing Challenge.

## Prerequisites

- **Go 1.20+**

## Setup

1.  Navigate to this directory:
    ```bash
    cd golang
    ```

2.  Download dependencies:
    ```bash
    go mod download
    ```

## How to Run

To run the main program:

```bash
go run main.go
```

## Development

### Formatting

We use `gofmt` for code formatting. To format your code:

```bash
go fmt ./...
```
