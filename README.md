# Tree Packing Challenge

This repository contains a solution to the Tree Packing Challenge, a Kaggle competition focused on optimizing tree packing problems.

## Directory Structure

- `python/`: Contains the Python implementation for visualizing results.
- `golang/`: Contains the Go implementation of the core solution.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Go**: Version 1.20 or higher. [Download Go](https://go.dev/dl/)
- **Python**: Version 3.13 or higher. [Download Python](https://www.python.org/downloads/)
- **Poetry**: For Python dependency management. [Install Poetry](https://python-poetry.org/docs/#installation)

## Getting Started

### 1. Clone the Repository

```bash
git clone <repository-url>
cd tree-packing-challenge
```

### 2. Set up the Python Environment

Navigate to the `python` directory and install dependencies:

```bash
cd python
poetry install
```

### 3. Set up the Go Environment

Navigate to the `golang` directory and download modules:

```bash
cd ../golang
go mod download
```
